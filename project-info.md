## 1. What is requirements.txt in python?
    A requirements.txt file in Python is used to list all the dependencies (packages and their versions) that your project needs. This makes it easy to share your project with others or deploy it, since they can install everything with a single command.

## To install all dependencies listed in the file requirements.txt:

    ``` bash
        pip install -r requirements.txt
    ```
